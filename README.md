## STAR WARS API

This STAR WARS API project was developed with [Loopback](https://loopback.io).

## Installation

```bash
# Get the project
git clone git@github.com:patilreena/loopback-auth0-starwars-api.git starwarsapi


# Change directory
cd starwarsapi

# Install the dependencies
npm install

# Run your app
node .
```
